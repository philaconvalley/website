## What is Open Source?

Open source means the code is publicly available for anyone to view, use, modify, and share. This website is open source!

## Making Your First Contribution

1. Find a project with a `good first issue` label
2. Fork the repository
3. Make your change
4. Open a pull request

That's it — you're now an open source contributor.
