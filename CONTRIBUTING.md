# Contributing to PhilaCon Valley

Thank you for your interest in contributing! We welcome everyone — whether you write code, design, create content, or just have ideas.

For detailed guides, see our docs:

- **[Getting Started](docs/getting-started.md)** — Set up the project on your computer
- **[Contributing Guide](docs/contributing.md)** — How to submit changes (code workflow, PR process, guidelines)
- **[Adding Content](docs/adding-content.md)** — Add a project or resource (no coding required)
- **[Architecture](docs/architecture.md)** — How the codebase is organized
- **[Design System](docs/design-system.md)** — Brand colors, fonts, and component styles

## Quick Links

- [Open an issue](https://github.com/philaconvalley/website/issues/new/choose) — Report bugs, suggest features, or submit content
- [View open issues](https://github.com/philaconvalley/website/issues) — Find something to work on
- [Code of Conduct](./CODE_OF_CONDUCT.md) — Please read before participating

## Questions?

- **Slack**: Join and ask in the #website channel
- **Email**: waskar@philaconvalley.com
- **GitHub**: Open a discussion or comment on an issue

Every contribution matters — from fixing a typo to building a feature. We appreciate you!
