# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email **waskar@philaconvalley.com** with:

- A description of the vulnerability
- Steps to reproduce it
- The potential impact

We will acknowledge your report within 48 hours and work with you to resolve the issue before any public disclosure.

## Scope

This policy applies to the PhilaCon Valley website codebase and its deployment at [philaconvalley.com](https://philaconvalley.com).

## Supported Versions

We only support the latest version deployed to production. Please ensure you are testing against the live site or the latest `main` branch.
